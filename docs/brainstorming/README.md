# Brainstorming

Explorations, spikes, and hypotheses. Files here are **non-binding** and **disposable** — delete or archive once resolved.

## Naming convention

`YYYY-MM-DD-<slug>.md`

## What belongs here

- "What if we did X?" explorations
- Spike results (time-boxed experiments)
- Options analysis before a spec is written
- Half-formed ideas that need more thought

## What does NOT belong here

- Decisions (those go in `docs/specs/`)
- Tasks (those go in `docs/todos/`)
- Stable facts (those go in `docs/knowledge/`)
