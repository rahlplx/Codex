Persistent cross-session state. Agents append to these files; they are never deleted.

## Format

Each entry uses this structure:

```markdown
## YYYY-MM-DD — <title>
<One-paragraph summary of what was learned or decided>
```

## Files

- `architecture.md` — System-level decisions and structural insights
- `api-quirks.md` — External API gotchas, rate limits, undocumented behavior
- `lessons.md` — Mistakes made and how they were resolved
