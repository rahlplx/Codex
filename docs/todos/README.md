# TODOs

Structured task tracking.

## Files

- `BACKLOG.md` — Prioritized active tasks and archive of completed tasks.

## How to use

- Add new tasks to BACKLOG.md under "Active" with a priority (P0=urgent, P1=high, P2=normal, P3=low).
- When a task is done, move it to the "Done" table with the completion date.
- For large features, link to the relevant `docs/features/` file.
- Do not put implementation details here — those belong in the feature or spec doc.
